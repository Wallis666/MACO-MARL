# 论文阅读笔记

存放与本项目相关的论文阅读记录。每篇论文一个 Markdown 文件。

## 命名规范

文件名格式：`YYYYMMDD-简短标识.md`

示例：`20260401-dreamerv3.md`、`20260405-m3w.md`

## 笔记模板

每篇论文的阅读笔记建议包含以下内容：

```markdown
# 论文标题

- **作者**：
- **发表年份 / 会议**：
- **论文链接**：

## 核心思想

用 2-3 句话概括论文要解决的问题和提出的方法。

## 与本项目的关系

说明该论文的哪些思路、模块、技术可以被本项目借鉴或复用。

## 关键方法

记录论文的核心算法、架构或公式。

## 实验结论

记录与本项目相关的实验结果和发现。

## 个人备注

记录自己的疑问、想法或改进思路。
```
